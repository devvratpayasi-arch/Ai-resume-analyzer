# 📄 AI Resume Analyzer with Career Prediction

A smart Resume Analysis web application built using **Python**, **Streamlit**, and **PDFPlumber**. The system analyzes uploaded resumes, predicts suitable career roles, calculates ATS scores, identifies missing skills, and provides recommendations for improvement.

## 🚀 Features

* Upload and analyze PDF resumes
* Predict career roles:

  * AI Engineer
  * Data Scientist
  * Web Developer
  * Backend Developer
* Calculate ATS Resume Score (0–100)
* Detect technical skills from resumes
* Identify missing skills for target roles
* Display career match rankings
* Generate resume improvement suggestions

## 🛠️ Tech Stack

* Python
* Streamlit
* PDFPlumber
